# react-newsapi

## development environment
#### Node v6.14.1
#### Npm v5.6.0

## Get Started
### Get code
```bash
git clone https://github.com/AgeeHuang/react-newsapi.git
```

### After clone
```bash
cd react-newsapi
npm install / yarn install
```

## Client side
- run dev client
```bash
npm run dev-client
```
- visit
    -	http://localhost:9000
